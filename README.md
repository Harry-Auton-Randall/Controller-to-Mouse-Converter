This program allows for the mouse cursor to be controlled by the right half of an Xbox controller. It's intended for when the user doesn't have enough surface space for an optical mouse.

The right stick control's the cursor's position, the right bumper is left click, and the right trigger is right click.\
The scroll wheel is operated with the A, B and Y buttons. Press/hold A to scroll down and B to scroll up. Y acts as the scroll wheel clicking.

The cursor's sensitivity and the program's framerate can both be changed via the Options menu, which is accessed by pressing O on the keyboard.
This, along with other functions triggered via the keyboard, can only be accessed while the X button on the controller is held down (the one with the A/B/Y buttons, not the big central one). This is to prevent triggering them accidentally while typing.

The controls are laid out this way so the user can comfortably operate them while holding the controller in their right hand, like a normal mouse.
